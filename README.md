# AI Business Card Scanner App

**Machine Test Submission - Adivinar Catalyst**

- **OCR**: Google ML Kit Text Recognition (on-device)
- **State Management**: Provider
- **HTTP**: Dio
- **Sheets**: Google Apps Script Web App (POST + GET)
- **Architecture**: Clean separation (core, features, services, models)

Setup:
1. flutter pub get
2. Paste Google Apps Script Web URL in core/constants/app_constants.dart
3. flutter run

All requirements fulfilled.
